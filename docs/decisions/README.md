# Decisiones tecnicas

Estos documentos registran decisiones que condicionan cambios futuros. No son
un diario: se crea una entrada cuando hubo alternativas razonables y conviene
conservar el motivo de la eleccion.

## Indice

- [0001 - Aplicacion local-first sin backend](0001-local-first.md)
- [0002 - Seleccion de ejercicios dentro de cada sesion](0002-session-exercise-selection.md)

## Formato

Cada decision incluye:

- estado;
- fecha;
- contexto;
- decision;
- alternativas consideradas;
- consecuencias.

Si una decision se sustituye, no se borra. Se marca como reemplazada y se enlaza
la nueva.
